Quick Start Guide

Get CLLOVA AI OS running in under 5 minutes.

System Requirements

  • Linux (Ubuntu 20.04+ recommended)
  • 4GB RAM minimum
  • Python 3.8+

Installation

Run this one-line installer:

curl -sSL https://install.cllova.ai | bash

First Run

cllova setup  # Follow the interactive guide
cllova start  # Launch the AI service

Voice API Reference

Programmatically interact with CLLOVA's voice interface.

POST /api/voice-command

Submit audio for processing:

{
  "audio": "base64_encoded_wav",
  "language": "en-US",
  "device_id": "optional_device_identifier"
}

Response:

{
  "text": "transcribed_command",
  "intent": "play_music",
  "confidence": 0.92
}

GET /api/voice-synthesis

Generate speech from text:

/api/voice-synthesis?text=Hello+World&voice=female-02

Troubleshooting Guide

Common Issues

Microphone Not Detected

Solution:

# List audio devices
arecord -l

# Set default device in config
echo "default_mic = hw:1,0" >> ~/.cllova/config.ini

Skills Not Loading

Debug steps:

cllova skills --verbose
journalctl -u cllova -f  # View live logs

Getting Support

For urgent issues, contact support@cllova.ai with your logs:

cllova diagnostics > diagnostics.txt
